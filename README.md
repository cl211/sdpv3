# sdpv3
Le site de prom's, le vrai !

## Update dependencies
```js
npm install
bower install
grunt wire
```

## Install dependencies
```js
bower i --save angular
npm i --save express
npm i --save-dev broccoli
```

## Run
```js
node server.js
```

## Wire
```js
grunt wire
```

## Build
```js
rm -rf dist && broccoli build dist
```
